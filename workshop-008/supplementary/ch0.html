IT 기술 전체 지도 (20분)

비전공자가 컴퓨터 전공 수준의 큰 그림을 20분에
보조 영상 1 개 · ~20분
1줄 핵심
비전공자가 컴퓨터 전공자 수준의 전체 그림을 20분에 잡는다. 탑다운 접근 = 숲 먼저 나무 나중.
요약
3 포인트
  1. 전공자 vs 비전공자 차이 — 전공자 = 소프트웨어 공학 + OS + 네트워크 + DB + 클라우드 체계적 학습. 비전공자 = 언어부터 시작 → 그림 없음.
  2. 탑다운 학습법 — 웹 / 앱이 어떻게 만들어지는지 전체 그림 먼저 → 필요할 때 세부 공부. 기술로트 PDF 목차 = 이 순서.
  3. 전체 지도 흐름 — FE (HTML / CSS / JS) → BE (Node / Python) → DB (MySQL) → 인프라 (서버) → 클라우드 (AWS). 순서 중요.
