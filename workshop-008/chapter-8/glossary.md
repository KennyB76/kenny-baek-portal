# Chapter 8 핵심 용어

| 용어 | 정의 | Kenny 비유 |
|------|------|-----------|
| 온프레미스 | 자체 서버실에 물리 서버 직접 운영 | 자가 주방 (내 집에서 요리) |
| 클라우드 | 인터넷으로 서버 자원 임대 | 배달 서비스 구독 |
| 가상화 | 물리 서버 1대를 여러 가상 서버로 분할 | 하나의 건물 = 여러 사무실 임대 |
| VM | Virtual Machine. 가상 컴퓨터 | 독립 사무실 (칸막이) |
| 하이퍼바이저 | VM 생성/관리 소프트웨어 | 건물 관리인 |
| IaaS | 인프라 임대. VM+네트워크+스토리지 | 빈 사무실 임대 (인테리어는 직접) |
| PaaS | 플랫폼 임대. OS+미들웨어 포함 배포 환경 | 인테리어 완료 사무실 (바로 업무 가능) |
| SaaS | 완성 소프트웨어 서비스 | 서비스드 오피스 (회의실+프린터 포함) |
| AWS | Amazon Web Services. 1위 클라우드 | 세계 최대 임대 빌딩 |
| GCP | Google Cloud Platform | 구글 임대 빌딩 |
| Azure | Microsoft 클라우드 | MS 임대 빌딩 |
| EC2 | AWS 대표 VM 서비스 (IaaS) | 빈 사무실 |
| Lambda | AWS 서버리스 함수 실행 서비스 | 프리랜서 (호출 시만 일함) |
| 서버리스 | 서버 관리 없이 함수 단위 실행 | 필요할 때만 알바 고용 |
| 오토스케일링 | 트래픽에 따라 서버 자동 추가/제거 | 손님 많으면 알바 추가 |
| Vercel | Next.js 최적화 PaaS 배포 플랫폼 | 완성형 사무실 + IT 지원 포함 |
| Docker | 컨테이너 기반 앱 패키징 도구 | 이삿짐 박스 (어디서나 설치) |
| Kubernetes | 컨테이너 오케스트레이션 플랫폼 | 이삿짐 박스 배달 총괄 |
| 용량산정 | 온프레미스 서버 크기 결정 작업. 수개월 소요 | 자가 주방 크기 결정 (잘못 짓기 어려움) |
| CapEx | Capital Expenditure. 서버 구매 선투자 | 집 사기 |
| OpEx | Operational Expenditure. 사용량 과금 | 월세 내기 |
