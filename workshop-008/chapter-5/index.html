Section A
클라이언트-서버 모델

클라이언트는 요청(Request)을 보내는 주체(브라우저, 앱)이고, 서버는 응답(Response)을 돌려주는 주체다. 인터넷의 모든 통신은 이 요청-응답 패턴을 기반으로 한다. 서버는 24시간 대기하며 포트(Port)를 열어 클라이언트의 요청을 수신한다.

Kenny (클라이언트/브라우저) → HTTP Request (GET /page.html) → DNS 조회: domain → IP → 서버 (IP:80/443 포트에서 대기) ← HTTP Response (200 OK + HTML)

서버jarvis (응답 주체)

Section B
HTTP / HTTPS

HTTP(HyperText Transfer Protocol)는 웹 통신 프로토콜이다. 텍스트 기반, 무상태(Stateless)가 특징이다. HTTPS는 HTTP + TLS(암호화)로, 데이터를 암호화해 중간자 공격을 방지한다. 상태 코드(Status Code)로 응답 결과를 전달한다.

상태 코드의미예시
200 OK성공페이지 정상 로드
301 Redirect영구 이동http → https 리다이렉트
401 Unauthorized인증 필요로그인 안 한 상태
403 Forbidden권한 없음접근 거부
404 Not Found리소스 없음잘못된 URL
500 Server Error서버 오류백엔드 버그
Section C
DNS / IP

IP 주소는 인터넷상의 컴퓨터 주소(숫자). DNS(Domain Name System)는 도메인 이름을 IP 주소로 변환하는 전화번호부다. 브라우저에 URL을 입력하면 DNS 조회 → IP 확인 → 서버 접속 순으로 진행된다. Cloudflare(1.1.1.1), Google(8.8.8.8)이 대표적인 퍼블릭 DNS다.

브라우저 주소창: newtonschool.net → OS가 캐시 확인 → DNS Resolver (1.1.1.1) 에 쿼리 → Root NS → TLD NS (.net) → Authoritative NS → IP: 185.x.x.x 반환 → TCP 연결 → HTTPS 핸드셰이크 → 페이지 로드

DNSMCP tool 라우팅 (이름 → 실제 tool)
IP 주소Tailscale IP (100.109.224.71 = homedesktop)
도메인newtonschool.net, newtonrealty.org
Section D
TCP / UDP

TCP(Transmission Control Protocol)는 신뢰성 있는 전송을 보장한다. 3-way Handshake로 연결을 확립하고, 패킷 손실 시 재전송한다. UDP(User Datagram Protocol)는 빠르지만 신뢰성을 보장하지 않는다. 스트리밍, 게임, DNS 쿼리에 사용된다.

항목TCPUDP
신뢰성보장 (재전송)미보장 (최선 전달)
속도느림 (오버헤드)빠름
연결3-way Handshake비연결
사용처HTTP, 파일 전송, 이메일동영상 스트리밍, 게임, DNS
NewtonHQ 대응jarvis API call (결과 보장)TTS 실시간 스트리밍
Section E
CDN / 프록시

CDN(Content Delivery Network)은 전 세계 엣지 서버에 콘텐츠를 캐싱해 사용자와 가장 가까운 서버에서 응답한다. 속도 향상 + 오리진 서버 부하 감소. 프록시(Proxy)는 클라이언트와 서버 사이에서 요청을 중계한다. 캐싱, 보안, 로드밸런싱에 활용된다. Cloudflare, AWS CloudFront가 대표적인 CDN이다.

Quiz
  1. 클라이언트-서버 모델에서 "무상태(Stateless)"란 무엇을 의미하며, 이를 보완하기 위해 쿠키/세션을 사용하는 이유를 설명하라.
  2. 브라우저에 https://newtonschool.net을 입력했을 때 일어나는 과정을 DNS → TCP → TLS 순으로 설명하라.
  3. HTTP 상태 코드 200, 301, 401, 404, 500의 의미를 각각 한 줄로 설명하라.
  4. TCP와 UDP의 주요 차이점을 신뢰성, 속도, 사용 사례 관점에서 비교하라.
  5. CDN을 사용하면 웹사이트 성능이 어떻게 개선되는지 설명하고, Newton 포탈에 CDN 적용이 필요한 이유를 말하라.
