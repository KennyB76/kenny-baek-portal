# 보조 - 프론트 / 백 / DB 20분 완성

## 1줄 핵심

웹 서비스 = 프론트(화면) + 백(로직) + DB(데이터) 3계층. 각각 다른 기술 스택. 연결 고리 = API.

## 요약 (3 포인트)

1. 프론트엔드 - HTML/CSS/JS 기본. React/Vue/Angular 프레임워크. 브라우저에서 실행. 사용자가 보는 모든 것.
2. 백엔드 - Node.js/Python/Java. 비즈니스 로직. DB 접근. API 응답 생성. 사용자 눈에 안 보임.
3. DB - MySQL/PostgreSQL (RDB) / MongoDB (NoSQL). 백엔드를 통해서만 접근. 직접 접속 X.
