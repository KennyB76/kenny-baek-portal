# Part1 - App vs Web, Java vs Python, OOP

## 5 손가락 요약

1. 탑다운 학습법 - 전체 지도 먼저, 세부 나중. 언어보다 개념 우선.
2. 언어 진화 계보 - C -> C++ -> Java -> Python/Node.js. OOP 계승.
3. OOP 핵심 3개 - Class (설계도), Object (실체), Inheritance (물려받기).
4. Frontend 3역할 - UI 구성 (HTML/CSS), 이벤트 처리 (JS), 서버 데이터 연결 (API call).
5. React / React Native - 웹과 앱 동시 개발. 컴포넌트 재사용. 비전공자도 접근 가능.

## 주요 용어 + 정의

| 용어 | 정의 |
|------|------|
| OOP (객체지향) | 현실 사물을 Class/Object 로 모델링하는 프로그래밍 패러다임 |
| Class | 비행기 게임 설계도. 속성(속도/HP) + 메서드(이동/공격) 정의 |
| Object | Class 를 실체화한 인스턴스. my_plane = Airplane() |
| Inheritance | 자식 클래스가 부모 속성/메서드 그대로 물려받음. C++ -> Java 계승 패턴 |
| Frontend | 브라우저/앱 화면 레이어. HTML(구조) + CSS(스타일) + JS(동작) |
| React | Facebook 개발 UI 라이브러리. 컴포넌트 단위 재사용 |
| React Native | React 코드로 iOS/Android 앱 빌드. 한 코드베이스 양플랫폼 |
