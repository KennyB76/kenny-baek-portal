# 보조 - Web / WebApp / Hybrid / Native 앱 유형

## 1줄 핵심

서비스 만들기 전 앱 유형 선택이 핵심. 웹/웹앱/하이브리드/네이티브 = 목적 + 예산 + 개발 속도 기준.

## 요약 (3 포인트)

1. 웹 - 콘텐츠 중심 서비스. HTML/CSS/JS. 브라우저로만 접근. 빠른 배포 (Vercel/Netlify).
2. 웹앱 / 하이브리드 - 앱처럼 보이지만 안은 웹 기술. React Native / Flutter. 한 코드 → iOS/Android.
3. 네이티브 - 각 플랫폼 전용. iOS = Swift / Android = Kotlin. 최고 성능. 개발 비용 2배.
