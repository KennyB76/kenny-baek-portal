# Chapter 7 핵심 용어

| 용어 | 정의 | Kenny 비유 |
|------|------|-----------|
| 아키텍처 | 시스템 전체 구성 설계 | 건물 설계도 |
| 3-Tier | Web(화면)+WAS(로직)+DB(데이터) 3계층 | 1층 매장/2층 주방/3층 창고 |
| 프레젠테이션 레이어 | 사용자 화면 담당 | 매장 (손님 응대) |
| 비즈니스 레이어 | 비즈니스 로직 처리 | 주방 (요리) |
| 퍼시스턴스 레이어 | 데이터 영구 저장 | 창고 (재료 보관) |
| 로드밸런서 | 트래픽을 여러 서버로 분산 | 계산대 여러 개 |
| 리버스 프록시 | 클라이언트 앞단에서 요청 중개 | 안내 데스크 |
| API Gateway | MSA 진입점. 인증/라우팅 통합 | 경비실 (신원 확인 후 안내) |
| CDN | 전 세계 정적 파일 분산 캐시 | 전국 편의점 재고 |
| 오토스케일링 | 트래픽 증가 시 서버 자동 추가 | 알바 자동 추가 고용 |
| 수평 확장 | 서버 대수 증가 | 계산대 추가 |
| 수직 확장 | 서버 사양 업그레이드 | 계산원 실력 향상 |
| 메시지 큐 | 서비스 간 비동기 통신 버퍼 | 주문서 대기열 |
| Circuit Breaker | 장애 서비스 자동 차단 | 두꺼비집 (과부하 차단) |
| 서비스 메시 | MSA 서비스 간 통신 인프라 레이어 | 사내 내선 전화망 |
| Nginx | 고성능 웹서버 + 리버스 프록시 | 다기능 안내 데스크 |
| 헬스체크 | 서버 정상 작동 여부 주기적 확인 | 직원 출근 체크 |
| 블루-그린 배포 | 두 환경 전환으로 무중단 배포 | A 계산대 → B 계산대 이동 |
| 카오스 엔지니어링 | 의도적 장애 주입으로 시스템 복원력 테스트 | 소방훈련 |
| SLA | Service Level Agreement. 서비스 가용성 약속 (예: 99.9%) | 영업시간 보장 계약 |
