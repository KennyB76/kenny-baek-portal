App vs Web vs Java vs Python

OOP 와 언어 진화 — 비전공자 탑다운 접근
Part1 영상 기반 · ~30분 · 2026 봄
1줄 핵심
탑다운 학습법으로 OOP 와 언어 진화 계보를 한 번에. Java/Python 선택 기준 + Frontend 3 역할 + React 컴포넌트 재사용까지 — 비전공자도 30분에 큰 그림.
5 손가락 요약
핵심 다섯 줄
  1. 탑다운 학습법 — 전체 지도 먼저, 세부 나중. 언어보다 개념 우선.
  2. 언어 진화 계보 — C → C++ → Java → Python / Node.js. OOP 계승 흐름.
  3. OOP 핵심 3개 — Class (설계도), Object (실체), Inheritance (물려받기).
  4. Frontend 3 역할 — UI 구성 (HTML/CSS), 이벤트 처리 (JS), 서버 데이터 연결 (API call).
  5. React / React Native — 웹과 앱 동시 개발. 컴포넌트 재사용. 비전공자도 접근 가능.
