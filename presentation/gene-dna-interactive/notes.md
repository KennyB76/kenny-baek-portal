## 2. Gene Expression & Epigenetics

- 같은 DNA → 다른 세포 (피부/뇌/간) = 어떤 gene 이 켜지느냐의 차이
- 후성유전학 (Epigenetics): DNA 서열을 바꾸지 않고 발현을 조절하는 메커니즘
  - **DNA 메틸화 (Methylation)**: 유전자 앞에 메틸기 부착 → 해당 유전자 침묵
  - **히스톤 변형 (Histone modification)**: DNA가 감긴 단백질 실패 조임 → 유전자 열림/닫힘
- 환경 / 스트레스 / 식이 / 사회적 관계 → 메틸화 패턴 변화 가능

출처: PMC6611551 / ScienceDirect

## 3. 사회 환경 → 유전자 발현 (정신과 fit 자료)

**NIH 연구 (PMC4021821)**:
- 어머니의 돌봄 방식 (licking/grooming 실험) → NR3C1 유전자 메틸화 패턴 직접 변화
- 따뜻한 양육 → 스트레스 반응 완화 / BDNF 증가 → 해마 발달 향상
- 방임/학대 → BDNF 메틸화 증가 → 스트레스 반응성 강화 + 기억 장애
- 이 효과는 **다음 세대 (grand-offspring)** 까지 전달 가능 (transgenerational)

**Nature Communications Biology (2021)**:
- 부모-자식 간 메틸화 패턴 상관관계 존재 BUT
- 환경이 달라지기 시작하면 그 패턴도 갈라짐
- 임신 중 / 출생 직후 = 후성유전적 각인의 결정적 시기

## 4. Kenny 비유 정합성 평가

| 비유 | 과학 근거 | 판정 |
|---|---|---|
| "발화된 gene 만 나옴" | gene expression 개념과 일치 | 정확 |
| "DNA = discrete" | 4염기 / codon 단위 이산 구조 | 정확 |
| "발현 = continuous" | 메틸화 수준·히스톤 조임 = 연속값 | 정확 |
| "파동 간섭 비유" | DNA 양자 파동 방출 연구 존재 (Nature 2024) / 직접 일치는 아님 | 시적 비유로 정합 |
| "부모→자식 예측 불가" | 후성유전적 재프로그래밍 + 환경 상호작용 → 비선형 | 연구 뒷받침 |

**결론**: Kenny 의 "discrete + continuous / wave interference" 비유는
- DNA 계층 (discrete) vs 발현 계층 (continuous) 구분으로서 과학적으로 정합
- 파동 비유는 시적 비유 — 완전한 엄밀성은 없으나 직관 전달에 유효
- 부모→자식 예측 불가 = 후성유전학 연구가 직접 뒷받침
