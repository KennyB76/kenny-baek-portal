### 2. 차원 사다리 구조

Kenny 통찰 순서:

```
3D 연속 (돌 + 물 + 공기)
    ↓  [평면 투영]
2D 중첩 (Huygens 파동 간섭 무늬)
    ↓  [측정 / 관측]
digital 점 (픽셀, 샘플링)
    ↓  [염기서열 해독]
ATGC (DNA 네 글자)
    ↓  [우리가 아직 모르는 것]
더 깊은 layer (분자 진동 / 양자 상태 / ?)
```

---

### 3. 핵심 비유 3가지

#### A. 돌 던지기 (§ I)

- 돌이 닿는 순간: 압력(힘) + 깊이(공간) + 시간 = 3차원 연속
- 아무것도 잃지 않은 상태
- "우리가 측정하기 전" 을 상징

#### B. Huygens 간섭 (§ II)

- 수면 위 무늬 = 이미 평면 투영
- 두 파원이 만들어내는 밝고 어두운 줄무늬
- 정보는 남아 있지만 — 차원이 줄었다
- 참고: Huygens (1629-1695) 의 파동 이론. 모든 파면의 점은 새 파원이 된다.

#### C. 측정 = 픽셀화 (§ III)

- 사진을 찍는 순간 = 양자 측정의 비유
- 슬라이더 올릴수록 무늬 뭉개짐 = 측정 단위가 정보를 결정
- 우리의 도구가 우리가 보는 현실을 만든다
