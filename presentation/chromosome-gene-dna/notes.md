# Research Notes — Chromosome / Gene / DNA Hierarchy

> 정신과 선생님 발표 보조 자료 (2026-05-16).
> Chromosome ↔ gene ↔ DNA 위계를 5 단계 zoom 으로 시각화.

---

## 1. The Big Picture — Hierarchy

```
몸 (Body)
 └── 세포 (Cell) — 약 37조 개
       └── 핵 (Nucleus) — 세포 1개당 1개
             └── 염색체 (Chromosome) — 사람 = 46 개 (23 쌍)
                   └── DNA — 이중 나선 (double helix)
                         └── 유전자 (Gene) — DNA 위 기능 단위
```

---

## 2. 세포 (Cell)

- 사람 몸 = 약 37조 개의 세포
- 대부분 세포 안에 핵(nucleus) 존재
- 핵 안에 유전 정보 전체가 압축 보관

---

## 3. 핵 (Nucleus) → 46 개의 염색체

- 핵 안에 23 쌍(= 46 개)의 chromosome
- 23 쌍 중 22 쌍 = 상염색체 (autosome)
- 1 쌍 = 성염색체 (sex chromosome): XX = 여성, XY = 남성
- 각 쌍 = 어머니 23 + 아버지 23 에서 유래
- Karyotype = 46 개 염색체를 크기/번호 순으로 정렬한 사진

### 각 염색체 번호별 주요 결정 요소 (대표)

| 번호 | 결정하는 것 (대표) |
|-----|----------------|
| 1 | 가장 큰 염색체 — 수천 개 유전자 |
| 7 | 낭포성 섬유증 관련 (CFTR) |
| 11 | 헤모글로빈 (산소 운반) |
| 17 | BRCA1 (유방암 관련) |
| 21 | 다운 증후군 (3 배) |
| X | 여성 성염색체 |
| Y | 남성 결정 (SRY 유전자) |

---

## 4. 염색체 → DNA 응축 4 단계

DNA 는 약 2 m 이지만 핵(~6 µm)안에 들어가려면 10,000:1 압축 필요.

```
Level 1: 벗은 DNA (naked DNA)
         — 이중 나선, 약 2 nm 폭

Level 2: Nucleosome (뉴클레오솜)
         — 히스톤 단백질 8 개 (octamer) 에 DNA 가 1.65 바퀴 감김
         — "구슬 목걸이 (beads-on-a-string)" 형태
         — ~11 nm 폭

Level 3: 30-nm 크로마틴 섬유 (chromatin fiber)
         — nucleosome 이 더 촘촘히 코일
         — 50x 단축

Level 4: 루프 + 스캐폴드 → 염색체 (chromosome)
         — 분열기(metaphase) 에 최대 압축
         — 10,000x 단축
```

---

## 5. DNA 이중 나선

- 두 가닥 (strand) 이 서로 반대 방향으로 꼬임
- 4 염기:
  - A (Adenine / 아데닌) — T 와 짝
  - T (Thymine / 티민) — A 와 짝
  - G (Guanine / 구아닌) — C 와 짝
  - C (Cytosine / 사이토신) — G 와 짝
- 사람 전체 DNA = 약 30 억 쌍 (3 billion base pairs)

---

## 6. Gene (유전자)

- DNA 위에서 특정 기능(단백질 생산)을 담당하는 연속 segment
- 사람 유전자 수 = 약 20,000 ~ 25,000 개
- 유전자 = DNA 전체 중 약 1.5 % 에 해당
- Gene 켜짐/꺼짐 = 환경·경험·스트레스에 의해 조절 (epigenetics)

### 대표 유전자 예

| 유전자 | 결정하는 것 |
|-------|-----------|
| OCA2, HERC2 | 눈 색깔 (갈색 / 파랑) |
| FGFR3 | 키 (연골 성장판) |
| MC1R | 머리카락 색 (빨강 유무) |
| BRCA1/2 | 유방암 위험도 |
| SRY | 남성 성별 결정 |
